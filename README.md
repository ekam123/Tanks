# Tanks

### Local turned based game, switch players after firing projectile. Last one remaining wins.  

#
![tanks_screenshot](https://github.com/ekam123/Tanks/assets/19808642/6939f676-170f-410f-bc4b-a2da0db5ed29)

### Features
- Choose between 2-5 players with custom names
- Select between three projectile weapons
- Custom wind affect with alternating strength levels and direction affecting projectile motion 
- Health and Scoring System
- Alter Turret Angle and Projectile Speed
- Chaos Vehicles Plugin


All assets from unreal engine marketplace.
